## Project Overview  
This system represents a simple UI mock-up of an online event registration platform.  
It includes pages for:  
- User login  
- User registration  
- Event creation  
- Event listing  
- Basic admin dashboard  

This project does not include backend functionality.  
It is created only for SDLC workflow demonstration.
